# react-native-react-dialpad

Dialpad for react native

## Installation

```sh
npm install react-native-react-dialpad
```

## Usage


```js
import { ReactDialpadView } from "react-native-react-dialpad";

// ...

<ReactDialpadView color="tomato" />
```
